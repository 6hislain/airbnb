# Airbnb Clone

An airbnb clone built using Nextjs 13 and Mongodb (using Prisma), video: https://www.youtube.com/watch?v=c_-b_isI4vg

## Requirements

- node 18
- code editor: vscode
- web browser: chrom
- mongodb

## Installation

- `git clone https://github.com/6hislain/airbnb`
- `npm install`
- `npm dev`
